How to - Lär dig göra CSS-animeringar

Animerings-koden

För att skapa en animation används regeln @keyframes. Här anges vilka egenskaper och värden animeringen ska ha när den börjar och när den slutar genom sektionerna ”from” och ”to”. Det går även att specificera animeringen i procent, vilket möjliggör oändligt många skiftningar i animeringen. Genom att ange ett procenttal mellan 0 och 100 specificeras vad som ska hända när animeringen är laddad till det angivna procenttalet. I alla exemplen nedan använder vi:
@keyframes circle {
0% {background-color: #C42576; top: 100px; left: 0px}
100% {background-color: #F1B6DA; top:100px; left: 70%}
}
@keyframes circle {
0% {background-color: #C42576; top: 0px;}
100% {background-color: #F1B6DA; top: 200px;}
}

Element-koden

För att animeringen ska fungera måste animerings-koden slutligen kopplas till ett element. Detta gör man genom att ange animationens egenskaper i CSS-kodblocket för elementet eller klassen. I exemplen nedan har vi t.ex. kopplat animeringarna via klassen .circle som i HTML-dokumentet ligger i en div. Här skapas strukturen för animeringen genom olika egenskaper, de kommer vi gå igenom nu.

animation-name

animation-name, vilket är det som skrivs efter @keywords i animerings-koden, anges i elementet. Det är animation-name som gör att animations-koden och elementet kopplas ihop och tillsammans skapar en animering.

@keyframes circle {
0% {background-color: #C42576; top: 100px; left: 0px}
100% {background-color: #F1B6DA; top:100px; left: 70%}
}
@keyframes circle {
0% {background-color: #C42576; top: 0px;}
100% {background-color: #F1B6DA; top: 200px;}
}

.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
 animation-duration: 3s;
animation-iteration-count: infinite;
}

animation-duration

Denna egenskap anges i sekunder (s) eller millisekunder (ms) och är den som bestämmer längden på animeringen. Vad som är viktigt att komma ihåg är att animation-duration alltid måste specificeras eftersom defaultvärdet för denna egenskap är 0. Därför kommer animeringen inte köras om detta inte anges.


.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 10s;
 animation-iteration-count: infinite;
}

animation-delay

Även denna egenskap anges i sekunder (s) och millisekunder (ms). Denna egenskap sätter hur lång tid som ska gå innan animationen startar. Om ett negativt värde anges så kommer användaren uppleva att animationen satts igång i förtid. Om till exempel värdet -2s anges så kommer alltså 2 sekunder att kapas från början av animationen. Hovra över cirkeln nedanför för att starta animationen.


.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 3s;
animation-iteration-count: infinite;
animation-delay: 2s
}

animation-iteration-count

Genom denna egenskap anges hur många gånger animationen ska köras. Om animationen ska fortsätta köras så länge sidan är öppen anges värdet ”infinite”.


.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 3s;
animation-iteration-count: infinite;
}

animation-direction

Med den här egenskapen sätts ordningen som animationen ska köras i, alltså framlänges eller baklänges. De värden som kan anges är:

  • normal - Spelar animationen framlänges, detta är default-värdet.
  • reverse - Ger en animation som spelas baklänges.
  • alternate - Animationen spelas först framlänges och sedan baklänges.
  • alternate-reverse - Animationen spelas först baklänges och sedan framlänges.

.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 3s;
animation-iteration-count: infinite;
animation-direction:alternate
}

animation-timing-function

Detta är en egenskap som anger animationens fartkurva. De värden som kan anges är:

  • ease - Ger animationen en långsam start, snabbt mittenparti och långsam avslutning. Detta värde är default.
  • linear - Detta ger en animation som har samma fart från start till slut.
  • ease-in - Animationen får en långsam start.
  • ease-out - Ger animationen ett långsamt slut.
  • ease-in-out - Animationen får en långsam start och långsamt slut.
  • cubic-bezier(n,n,n,n) - Låter dig definiera dina egna värden med cubic-bezier-funktionen

.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 3s;
animation-iteration-count: infinite;
animation-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55)
}

animation-fill-mode

Med den här egenskapen anges vad som ska hända innan och efter att animeringen körs. Möjliga värden är:

  • none - Detta är default-värdet och innebär att ingen animations-styling kommer att tillämpas innan eller efter att animationen körs.
  • forwards - Med detta värde kommer elementet att behålla den styling som det hade i slutet av animationen.
  • backwards - Det här värdet gör att elementet får den styling som det hade i början av animeringen och behåller det under animation-delay-perioden
  • both - Här kommer animationen att tillämpa både forwards- och backwards-värdena vilket gör att animeringen förlängs både i början och slutet.

.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 3s;
animation-fill-mode: forwards;
}

animation-play-state

Den här egenskapen specificerar om animationen ska köras eller pausas. De värden som kan anges är:

  • running - Detta värde är default och innebär att animationen körs.
  • paused - Detta gör att animationen pausas
  • initial - Sätter värdet till dess default.
  • inherit - Gör att egenskapen får samma värde som dess förälder

Hovra över cirkeln (eller tryck för mobil) för att pausa den.


.circle {
width: 70px;
height: 70px;
border-radius: 100%;
position: relative;
animation-name: circle;
animation-duration: 10s;
animation-iteration-count: infinite;
animation-play-state: paused
}

Flera värden i en egenskap

Det är även möjligt att ange flera värden för en egenskap, detta görs om flera animationer ska deklareras i samma regel.
Genom kommaseparerade värden kan du ange namnen på samtliga keyframes som ska användas.
Övriga värden kan sedan separeras på samma sätt för att ange olika värden per regel.
Om färre värden anges än antalet regler som ska ingå så loopas värdena, alltså tilldelas det första värdet igen och följs sedan av de andra i kronologisk ordning tills alla regler har fått ett värde.
