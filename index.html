Om CSS-animeringar

CSS-animeringar är ett enkelt sätt att göra din sida mer levande och tilltalande.

En av fördelarna med att använda CSS-animering framför Javascript är att animeringen fungerar även om användaren surfar med Javascript avstängt.

En annan fördel med CSS-animeringar är att de generellt fungerar bra även om sidan laddas långsamt. Detta till skillnad från Javascript-animeringar som kan få sämre kvalité ifall prestandan är låg.

Första webbläsare ut att stödja CSS animations var Firefox när de i juni 2011 släppte Firefox 5. Idag stöds modulen av de etablerade webbläsarna - som Google Chrome, Microsoft Edge, och Safari.
